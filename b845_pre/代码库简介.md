# field-calc 代码库简介

field-calc 是一个基于 Python 的命令行工具，专门用于根据高斯定律计算点电荷的电通量和电场强度。该工具旨在简化电磁学计算，让物理学生、教师和工程师能够快速获得精确结果，无需手动应用公式或记忆常数。

## 核心功能
- **电通量计算**：计算任意点电荷的总电通量（Φ = q/ε₀）
- **电场强度计算**：计算电场强度（E = q/(4πε₀r²)）
- **灵活输出**：支持具体半径值（数值结果）或符号半径 "r"（通用表达式）
- **格式化显示**：采用科学记数法格式化输出，确保可读性

## 技术栈
- **编程语言**：Python 3.6+
- **核心依赖**：标准库 math 和 sys
- **扩展依赖**：sympy（计划集成以支持更复杂的符号计算）
- **打包工具**：setuptools
- **容器化**：Docker（基于 Python 3.12-slim 镜像）
- **测试框架**：pytest

## 部署与使用
- **安装方式**：通过 pip 安装（`pip install -e .`）
- **命令行接口**：`field-calc <charge_in_C> [radius_in_m|"r"]`
- **容器运行**：提供 `build_docker.sh` 和 `run_docker.sh` 脚本
- **跨平台支持**：支持 Linux、macOS、Windows（通过 Docker）

## 应用场景
- **教学演示**：物理课堂上的电磁学计算示例
- **作业验证**：学生验证计算结果的准确性
- **工程估算**：快速获取电场强度近似值
- **科研辅助**：简化实验数据处理流程

该工具将繁琐的物理计算转化为简单的命令行操作，提高了计算效率和准确性。
