# 代码库简介（Cipher Suite）

该代码库的目标是提供一个轻量的 Python 加密算法教学/示例包，聚焦两种经典替换密码：凯撒密码与维吉尼亚密码。它要解决的问题是：在不引入复杂依赖的情况下，给出可直接调用的 `encrypt`/`decrypt` 接口，用于对英文文本进行加解密，同时保持空格、标点等非字母字符不变。项目采用 Python 实现，使用 `setuptools`（`setup.py`）进行打包与安装，测试框架为 `pytest`，并提供 `Dockerfile` 以便在容器中构建与运行测试。核心算法分别位于 [`project/code/10/cipher/caesar.py`](project/code/10/cipher/caesar.py) 与 [`project/code/10/cipher/vigenere.py`](project/code/10/cipher/vigenere.py)，对应的用法与测试运行方式在 [`project/code/10/README.md`](project/code/10/README.md) 中给出。
